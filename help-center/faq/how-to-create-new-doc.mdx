## Creating a Document

To create a new document in Messi:

1. Go to your Dashboard
2. Click `New Document`
3. Enter a name and description
4. Click `Create`

## Where does it appear?

- The document shows up on your Docs home
- You can organize by folders or tags

## Tips

- Use clear names for easy search
- Assign a team owner if needed
