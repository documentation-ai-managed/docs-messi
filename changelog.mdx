## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for teams to edit docs simultaneously
- Introduced version history with diff viewer for tracking changes
- New embed blocks for videos and interactive components

## Improvements

- Enhanced search with fuzzy matching and full-text indexing
- Optimized rendering speed by `>40%` for large doc sites
- Better mobile responsiveness across all themes

## Bug Fixes

- Fixed export to PDF failing on pages with tables
- Resolved authentication issues during webhook callbacks
- Patched potential XSS in custom HTML blocks

</Update>

## How to Update

Follow these steps to upgrade Messi to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your workspace:
    
````bash
messi export --workspace my-docs --format zip
````
  </Step>
  <Step title="Update CLI" icon="download">
    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install messi@latest -g
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn global add messi@latest
````
      </Tab>
      <Tab title="Docker" icon="docker">
````bash
docker pull messidocs/messi:latest
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Migrate Config" icon="settings">
    Run the migration tool if upgrading from v1.0:
    
````bash
messi migrate --config config.json
````
  </Step>
  <Step title="Restart Server" icon="play">
    Restart your Messi instance and verify at `http://localhost:3000`.
  </Step>
</Steps>

<Callout kind="tip">
  Check the [upgrade guide](/quickstart#upgrading) for version-specific instructions.
</Callout>
