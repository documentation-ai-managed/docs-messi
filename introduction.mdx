## Overview

Messi provides a centralized platform for teams to create, organize, and maintain project documentation. You can build structured docs sites with Markdown, MDX, and custom components, all while collaborating in real-time. Messi handles versioning, search, and deployment, so you focus on content rather than infrastructure.

<Callout kind="info">
  Messi supports both public and private documentation spaces, making it ideal for open-source projects and internal wikis.
</Callout>

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your free account at `https://dashboard.example.com/signup`.
  </Step>
  <Step title="Create Space" icon="folder-plus">
    Set up a new documentation space for your project.
  </Step>
  <Step title="Add First Doc" icon="file-plus">
    Upload or write your `introduction.mdx` file.
  </Step>
  <Step title="Publish" icon="rocket">
    Deploy your site with one click.
  </Step>
</Steps>

Here is a simple example to embed a code snippet in your docs:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch your docs data
  const response = await fetch('https://api.example.com/docs');
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  # Fetch your docs data
  import requests
  response = requests.get('https://api.example.com/docs')
  docs = response.json()
  print(docs)
  ```
